## 문제점
1. 전체 테마를 파란색 계열로 변경하길 원함
2. 맥주 관리 페이지에서 맥주 리스트 순서를 변경할 수 있도록 버튼 했는데 이 기능이 작동하지 않음
3. 통계 페이지의 달력이 실제 달력과 날짜-요일이 일치하지 않음
4. 통계 페이지는 본 폴더의 3.png, 4.png 사진의 수준과 같이 다양한 정보를 보길 원함

## 해결방안
1. 파란색 테마 적용 🔵
- 전체 UI를 파란색 계열로 변경 (blue-600, indigo-600, cyan-600 등)
- 그라데이션과 액센트 컬러 모두 파란색 톤으로 통일
2. 맥주 순서 변경 기능 구현 ↕️
- 위/아래 화살표 버튼으로 맥주 순서 변경 가능
- 실시간으로 데이터베이스에 sort_order 업데이트
- 첫 번째/마지막 항목에서는 해당 방향 버튼 비활성화
3. 달력 날짜-요일 정확성 수정 📅
- getDay() 함수를 사용해 월 시작일의 요일 계산
- 빈 셀을 추가하여 실제 달력과 정확히 일치하도록 수정
- 날짜와 요일이 완벽하게 매칭됨
4. 통계 페이지 대폭 개선 📊
- 첨부해주신 이미지 수준으로 다양한 정보 제공:

- 월별 통계:
-- 기존 3개 지표 + 4개 추가 지표 (총 음주일, 최대 음주량, 총 알코올량, 평균)
-- 선호 맥주 랭킹 (1-3위, 메달 아이콘 포함)
-- 각 맥주별 소비량과 총 용량 표시

- 연별 통계:
-- 월별 상세 breakdown
-- 연간 요약 통계
-- 연간 선호 맥주 랭킹
